ABCDEcoffee

架空のコーヒーショップサイトを制作。（HTML・CSSを2か月程学習後に制作）

サイトURL
https://yuina-furuya.github.io/ABCDEcoffee/
制作範囲
デザイン・コーディング・githubへのアップ
制作期間
デザイン : 2週間 / コーディング : 2週間
制作ツール
Adobe XD : ワイヤーフレームの作成・デザイン
Visual Studio Code : HTML・CSSでコーディング
コンセプト
顧客に心から落ち着ける空間を届けたい。こだわりのコーヒー・スイーツを味わってほしい。
ターゲット
30代の会社員
制作目的
来客数を増やす。お店のメニューを知ってもらう。
工夫した点
落ち着いた空間を表現するため、ダークトーンのカラーを使用し、緑を配色しました。
ユーザが利用しやすいよう、ヘッダー・メニューアイコン・フッターから全てのページにつながるよう設計し、レスポンシブにも対応させました。
苦労した点
レスポンシブ対応に苦労しました。特にモバイル版の、メニューアイコンをクリックすると横からメニューが表示される部分は、学習サービスで扱われなかったため、ネットで検索・学習し、コーディングしました。
コンタクトフォームは、勉強不足で実際は運用できない状態です。今後学習を進め、実装したいと考えています。
